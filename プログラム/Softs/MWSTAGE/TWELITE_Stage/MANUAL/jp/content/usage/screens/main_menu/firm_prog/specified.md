# 指定

`Windows` `macOS` `Linux`

フォルダまたは.BINファイルをTWELITE STAGE APP の画面にドラッグ＆ドロップできます。
ドロップした対象のビルドや書き込みを行うときに選択します。

{% hint style="info" %}
.BINファイルをドロップしたときは、そのファイルが格納されるフォルダをドロップした場合と同様の振る舞いで、そのフォルダにあるファームウェア一覧が表示されます。
{% endhint %}
