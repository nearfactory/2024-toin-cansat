E_PKT

Corresponds to the following packet

Corresponds to the ASCII format output by the Parent Node of App_Wings.

Name Description
PKT_ERROR TwePacket does not contain meaningful data such as before packet interpretation or packet type cannot be identified.
PKT_TWELITE 0x81 command of the standard application App_Twelite is interpreted.
PKT_PAL Interpret serial format of TWELITE PAL
PKT_APPIO UART message of remote control application App_IO products/ TWE-APPS/App_IO/uart.html) interpreted by the remote control application App_IO.
PKT_APPUART Extended format of the serial communication application App_UART products/ TWE-APPS/App_Uart/mode_format.html) is interpreted.
PKT_APPTAG The UART message of the wireless tag application App_Tag is interpreted. The sensor specific part is not interpreted and the byte string is reported as payload.
PKT_ACT_STD Output format used in Act (Act) sample, etc.
