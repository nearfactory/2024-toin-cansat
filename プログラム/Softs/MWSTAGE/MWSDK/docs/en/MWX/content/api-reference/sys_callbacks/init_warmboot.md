# init\_warmboot()

{% hint style="danger" %}
Usually not used.
{% endhint %}



Called in re-initialization when the peripheral API is not initialized after returning from sleep.

{% hint style="info" %}
This function can be used to detect interrupt factors.
{% endhint %}

{% hint style="info" %}
This callback function definition is optional.
{% endhint %}
