System start-up

Flow at system start-up.

The E_EVENT_START_UP event is only raised for user event processing functions.

  • : Starting
  • : In-system processing
  • : or
  • : User defined event handling functions
  • E_EVENT_START_UP : The E_EVENT_START_UP event is raised to the user-defined event processing function before the system starts the main loop operation.
