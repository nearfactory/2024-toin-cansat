ファームウェア書換

出来上がった bin ファイルを書き換えるには、PC のシリアル接続と、TWELITE 無線モジュールのポート制御が必要になります。

TWELITE DIP と TWELITE R の組み合わせでは、書き換え制御を自動化するため処理が容易です。
