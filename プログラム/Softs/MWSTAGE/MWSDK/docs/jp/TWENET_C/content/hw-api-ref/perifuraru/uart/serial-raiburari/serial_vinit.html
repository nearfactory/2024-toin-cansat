SERIAL_vInit()

解説

UART （シリアルポート）の初期化を行う。

本関数では１つの UART ポートの初期化を行い SERIAL ライブラリによるアクセスを可能とします。

初期化時に SERIAL ライブラリが内部的に利用する FIFO バッファーのメモリ領域を静的に確保する必要があります。

同時に、出力用の vfPrintf(), vPutChar() を行うため tsFILE 構造体の初期化を行っておきます。

引数

型 名前 詳細
tsSerialPortSetup* psSetup UART の設定用の構造体。

戻り値

なし

サンプル

tsSerialPortSetup 参照。
