# 中継ネットのアドレス

中継ネット特有のアドレス指定です。

| 定義名                                  | 値            | 意味                             |
| ------------------------------------ | ------------ | ------------------------------ |
| `TOCONET_NWK_ADDR_PARENT`            | `0x80000000` | 親機                             |
| `TOCONET_NWK_ADDR_\ NEIGHBOUR_ABOVE` | `0x80000001` | ネットワークアドレス、直接接続している中継・親機       |
| `TOCONET_NWK_ADDR_BROADCAST`         | `0x8000FFFF` | ネットワークアドレス、ブロードキャスト（親機からの指定のみ） |
