# ネットワーク (NWK)

ネットワーク ビヘイビアは、IEEE802.15.4のMAC層のパケットの送受信に対して、アドレス定義、配送制御などを行います。

* [\<NWK\_SIMPLE>](nwk\_simple.md) - 非常にシンプルな簡易中継ネットワークです。
* [\<NWK\_LAYERED>](nwk\_layered.md) - レイヤー補助を用いた簡略化された木構造ネットワーク。(注：MWXは受信のみのパラントノードしかサポートしていません。）
