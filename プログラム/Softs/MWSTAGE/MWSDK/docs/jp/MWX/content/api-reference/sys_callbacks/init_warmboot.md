# init\_warmboot()

{% hint style="danger" %}
通常は使用しません。
{% endhint %}



スリープ復帰後、ペリフェラルAPIが初期化されない再初期に呼び出されます。

{% hint style="info" %}
この関数では割り込み要因の検出を行うことができます。
{% endhint %}

{% hint style="info" %}
このコールバック関数定義は省略可能です。
{% endhint %}
